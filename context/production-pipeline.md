# DOF Production Pipeline

## 完整流程概覽

```
Client Brief → Concept / Script → Script Breakdown → Pitch Deck → Call Sheet → Shoot → Post-Production → Delivery
```

---

## Pre-Production 階段

| 階段 | 產出 | 工具 |
|------|------|------|
| Client Brief | Brief document | Client 提供 |
| Concept / Script | Script document | Google Docs |
| Script Breakdown | Shot list、talent list、location list | Google Docs / Sheets |
| Pitch Deck / Storyboard | 客戶提案文件 | Canva / Shotflow Studio |
| Call Sheet | 拍攝日程表 | Google Docs / Sheets |

## 標準 Project Timeline（Corporate Video）

### Pre-Production（由 Shooting 倒推）

| 步驟 | 標準工時 | 備註 |
|------|---------|------|
| DOF 撰寫 Script（如需要） | 5 working days | 只有 DOF 寫 script 時計算 |
| Client Confirm Script | 2–3 wd | |
| 準備 Video Flow + Graphics Reference | 5–6 wd | |
| Client Feedback on Video Flow + Ref | 2–3 wd | |
| Confirm → Shooting | 3–4 wd | |

Style Frame（可同 shooting 平行進行）：
| Submit Style Frame | 4–5 wd after Confirm Ref | |
| Client Feedback on Style Frame | 2–3 wd | |
| Confirm Style Frame | 理想喺 1st Cut 前 4–5 wd | 如趕唔切可同 1st Cut 一齊交 |

### Post-Production 標準工時

| 里程碑 | 標準工時 | 最短工時 |
|--------|---------|---------|
| 1st Cut（still graphics） | 5 wd after shoot | 4 wd |
| Client Feedback | 3 wd | 1 wd |
| 2nd Cut（animated graphics） | 3–5 wd | 3 wd |
| Client Feedback | 3 wd | 1 wd |
| 3rd Cut | 3 wd | 2 wd |
| Client Feedback | 3 wd | 1 wd |
| Picture Lock → VO Recording buffer | 2–3 wd | 2 wd |
| Color Grading + Sound Mixing | 2 wd | 1 wd |

**整個後期大約一個月。**

## Post-Production 標準流程

### Cut 版本定義

| 版本 | 內容 |
|------|------|
| 1st Cut | Still graphics（靜態字卡）；或 placeholder「假字」+ AI guidetrack VO |
| 2nd Cut | Animated motion graphics + fine-tune color + SFX |
| 3rd Cut+ | 逐步收窄改動，直至 Picture Lock |
| After Picture Lock | VO recording（如需要）→ Subtitle → Final color grading → Audio mixing → Final Cut |

### Cut Submit 流程（每個 cut 一致）
1. Editor export 片
2. Upload YouTube（**Unlisted**）
3. Share link 俾 Client（WhatsApp + Email）
4. Client review → 提供 feedback
5. 導演整理 feedback → update Video Flow document

### 後期分工

| 角色 | 職責 |
|------|------|
| Editor | Video editing（剪接、footage 組合）|
| Motion Graphics | 字卡、on-screen text、motion graphics、lower thirds |
| Post-Pro Supervisor（Sohling） | 統籌後期分工、QC |

---

## Video Flow Document（後期核心 working document）

後期嘅 single source of truth。導演根據 script 建立，後期同事對住做。

**典型 columns：**
| Column | 內容 |
|--------|------|
| Section / Timecode | Part name 或 timecode range |
| Script / VO | 旁白、對白 |
| Visual Description | 畫面描述、shot type |
| Screenshot REF | 參考圖 |
| Graphics | 字卡、on-screen text 描述 |

- First Cut 之前靠 script flow，First Cut 後加入 timecode
- Client 改 script → document 同步更新

---

## Client Feedback 階段

- **1st Cut + 2nd Cut**：通常係 Working Level（同 Client 接頭人 align）
- **2nd Cut 或 3rd Cut**：Client 可能拿去做 Senior Approval（額外需要幾日至一星期）
- Senior Approval 時間視乎 Client，導演手動調整 timeline

---

## Google Calendar 排期

DOF 有共用 **DOF Internal** Google Calendar，所有同事可見。主要 milestone 應 mark 入去：
- Shooting
- 1st Cut submit date
- 2nd Cut submit date
- Picture Lock
- Final Output

**標準 Milestone Set：**
`Shoot` → `1st Cut` → `Client FB 1` → `2nd Cut` → `Client FB 2` → `3rd Cut` → `Picture Lock` → `Final Output`

Milestone 命名跟 `naming-conventions.md` 的規定。
