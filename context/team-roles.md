## 職責分工

### 導演（Director）負責：
- 接收 client brief，跟進前期製作
- 建立 Video Flow working document（後期核心 document）
- 整理每輪 client feedback 並更新 working document
- 最終 QC + deliver 成片
- 管理 Google Calendar milestone entries

### 後期組負責：
- **Editor（Yik / Katy）**：Video editing、footage 組合、timecode、color（初步）
- **Motion Graphics（Keith / Max）**：字卡、lower thirds、2D animation、infographics
- **Graphic Design（Kay）**：KV、poster、banner（static graphic）
- **Supervisor（Sohling）**：後期統籌、分工、QC、同導演對接

### 老闆（Ki）負責：
- Quotation 撰寫及審批
- 最終 invoice 發出
- 重要 client 關係維護

---

## 合作模式

### 導演 ↔ 後期
1. 導演建立 Video Flow document（Google Doc）
2. 後期組對住做，各自分工（Editor 做剪、MoGraph 做字卡）
3. 做完喺 Google Doc mark done
4. 導演 QC 成品
5. Client feedback → 導演整理 → update working document

### 老闆 ↔ 導演
- 老闆收 enquiry → assign 導演跟 project
- 導演接 project 通常獲得口頭 briefing
- 導演負責前期製作直至最終交付
